vscode 快捷键

有用的 VS Code 快捷键

Ctrl + P ：转到文件，您可以在 Visual Studio Code 中移动到打开的文件/文件夹的任何文件。
Ctrl + ` ：在 VS Code 中打开 terminal
Alt + Down：下移一行
Alt + Up：上移一行
Ctrl + D：将选定的字符移动到下一个匹配字符串上
Ctrl + Space：触发建议
Shift + Alt + Down：向下复制行
Shift + Alt + Up：向上复制行
Ctrl + Shift + T：重新打开最新关闭的窗口
