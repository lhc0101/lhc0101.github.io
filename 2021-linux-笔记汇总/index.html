npm 与 node 管理与升级

node

centos 下使用 yum 默认安装的 nodejs 版本为 v6.x。可以使用 n 命令来升级切换 nodejs 版本。

安装 n 命令
n 是一个 nodejs 管理工具，https://github.com/tj/n

$ npm install -g n
使用 n 安装 nodejs 版本
安装最新版本

安装指定版本

$ n node/13.11.0
切换 nodejs 版本
$ n

Use up/down arrow keys to select a version, return key to install, d to delete, q to quit
使用 node -v命令，查看当前版本，如果显示如下则表示切换成功。

v13.11.0
注意：切换之后，查看版本如果还是原来 v6.x 版本，需要设置一下 PATH
(通常这一步可以忽略的，先自行测试，如皋版本确实没用再进行以下步骤)

设置 PATH
查看 node 当前安装路径

编辑配置文件

$ vi ~/.bash_profile
将下边两行插入到文件末尾：

:wq 保存退出

执行 source 使修改立即生效

$ source ~/.bash_profile
再执行 node -v 命令查看版本是否切换成功。

npm

查看npm当前版本

npm -v

如果不是最新版本，运行指令

npm install -g npm

如果想更新到指定版本，运行指令

npm -g install npm@6.9.0
